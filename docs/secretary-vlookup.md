# 秘書向け：VLOOKUP で患者名を表示する手順

このアプリから Google スプレッドシートに送られるデータには **患者名は含まれません**（PID のみ）。
秘書の PC 上の **患者マスタ** と照合して、名前を表示します。

---

## VLOOKUP とは？

**2つの表を PID で照合する** Excel / スプレッドシートの機能です。

```
【共有シート】集計_患者          【秘書PC・ローカル】患者マスタ
  PID_A3F2...  移動15分  診療20分    PID_A3F2...  田中太郎
       ↓ VLOOKUP で照合 ↓
  田中太郎  移動15分  診療20分   ← 秘書の画面に表示
```

---

## 準備

### 1. 共有スプレッドシート（自動更新）

アプリが書き込むシート：

| シート名 | 内容 |
|---------|------|
| **記録** | 打刻ログ（PID・施設・イベント） |
| **集計_患者** | 今日の患者別時間（その日だけ上書き） |
| **集計_施設** | 今日の施設別時間（その日だけ上書き） |
| **患者別_履歴** | 日付×PID ごとの時間（過去分を蓄積） |
| **施設別_履歴** | 日付×施設 ごとの時間（過去分を蓄積） |

**集計_患者 / 集計_施設** … 今日の確認用（毎日上書き）  
**患者別_履歴 / 施設別_履歴** … 数か月分が溜まる（過去の日付は消えない）

### 2. 患者マスタ（秘書PC・ローカル Excel）

**Google には上げない** ファイルを秘書 PC に置きます。

| A列: 患者ID(匿名) | B列: 患者名 | C列: メモ |
|-------------------|------------|----------|
| PID_A3F2B1... | 田中 太郎 | ひまわり |
| PID_B7C8D2... | 佐藤 花子 | |

**新しい PID が出たら、秘書が1行追加するだけ** で OK です。

---

## Excel での設定例

### 集計_患者 を Excel で開く

1. Google スプレッドシート → **ファイル → ダウンロード → Microsoft Excel**
2. または Excel でスプレッドシートを開く（同期）

### 表示用シートに数式を入れる

`集計_患者` の E列が PID、G〜J列が時間の場合：

**F列（患者名）に：**

```excel
=XLOOKUP(E2, 患者マスタ!A:A, 患者マスタ!B:B, "【要登録】")
```

古い Excel の場合（VLOOKUP）：

```excel
=IFERROR(VLOOKUP(E2, 患者マスタ!A:B, 2, FALSE), "【要登録】")
```

- マスタに PID があれば → **患者名**
- なければ → **【要登録】** → マスタに追加

---

## 日常の流れ

```
1. 医師がアプリで記録
      ↓ リアルタイム
2. 共有スプレッドシートが更新（PIDのみ）
      ↓
3. 秘書が Excel で開く
      ↓ VLOOKUP / XLOOKUP
4. 患者名 + 時間が表示される
      ↓
5. 未知の PID → 患者マスタに1行追加（次回から自動表示）
```

---

## 複数秘書の場合

- **共有シート** — 全員が同じ URL を閲覧（PID のみなので問題少）
- **患者マスタ** — 各秘書 PC のローカル（または院内共有フォルダの Excel）

---

## 注意

| やる | やらない |
|------|---------|
| PID + 時間をクラウド共有 | 患者マスタを Google にアップロード |
| 秘書 PC で名前変換 | 発話内容（マスク済）を社外に |

発話内容列は「〇〇」にマスク済みですが、念のため秘書以外には共有しないでください。
