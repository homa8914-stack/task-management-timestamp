# 秘書向けセットアップガイド

医師のアプリから **PID（匿名ID）と時間だけ** が共有されます。  
**患者名はこのPC上の「患者マスタ」だけ** に保存してください（Google には上げない）。

---

## 全体の流れ

```
共有スプレッドシート（PID + 時間）
        ↓  VLOOKUP / XLOOKUP
患者マスタ（このPC・ローカル）
        ↓
「誰に何分」が見える
```

---

## ステップ1：スプレッドシートを開く

医師から **共有リンク** をもらい、ブラウザで開きます。

| シート | 見る用途 |
|--------|---------|
| **集計_患者** | 今日の患者別 |
| **集計_施設** | 今日の施設別 |
| **患者別_履歴** | 過去～今日（PID別・日付順） |
| **施設別_履歴** | 過去～今日（施設別・日付順） |
| **記録** | 詳細ログ（必要なときだけ） |

権限は **閲覧者** で十分です。

---

## ステップ2：患者マスタを作る（ローカル）

1. 付属の `患者マスタ.csv` を Excel で開く  
   （または新規 Excel → A1:C1 に `患者ID(匿名)` `患者名` `メモ`）
2. **自分のPC** に `患者マスタ.xlsx` として保存  
3. **Google ドライブにはアップロードしない**

| 患者ID(匿名) | 患者名 | メモ |
|-------------|--------|------|
| PID_A3F2B1... | 田中 太郎 | ひまわり |
| PID_B7C8D2... | 佐藤 花子 | |

### 新しい PID が出たとき

1. **患者別_履歴** の E列（患者ID）を見る  
2. マスタにない PID → **1行追加**（医師に毎回聞く必要はありません）  
3. 名前が分かればマスタに書く → 以降自動表示

---

## ステップ3：名前を表示する（Excel）

### 方法A：履歴シートを Excel で開く（おすすめ）

1. 共有スプレッドシート → **ファイル → ダウンロード → Microsoft Excel**
2. **患者別_履歴** シートを開く  
   （E列 = PID、G～J列 = 時間）
3. F列（患者名）の F2 に数式：

```excel
=XLOOKUP(E2, [患者マスタ.xlsx]患者マスタ!A:A, [患者マスタ.xlsx]患者マスタ!B:B, "【要登録】")
```

患者マスタを同じフォルダに置く場合：

```excel
=XLOOKUP(E2, 患者マスタ!A:A, 患者マスタ!B:B, "【要登録】")
```

古い Excel（XLOOKUP なし）：

```excel
=IFERROR(VLOOKUP(E2, 患者マスタ!A:B, 2, FALSE), "【要登録】")
```

4. F2 を下までコピー

### 方法B：Google スプレッドシート上（マスタは非共有）

1. 共有ファイルを **コピー** して自分のドライブに保存  
2. コピーしたファイルにだけ **「患者マスタ」** タブを追加（PID | 患者名）  
3. **患者別_履歴** の F2 に：

```excel
=XLOOKUP(E2, 患者マスタ!A:A, 患者マスタ!B:B, "【要登録】")
```

※ コピー版は秘書個人のもの。マスタ付きファイルを他人と共有しない。

---

## ステップ4：見やすくする（任意）

### 特定の患者だけ見る

**患者別_履歴** で E列（PID）または F列（患者名）を **フィルタ**

### 月ごとの合計（ピボット）

1. **患者別_履歴** を全選択  
2. **挿入 → ピボットテーブル**  
3. 行：`患者名`（または PID）  
4. 値：`合計時間（分）` の合計  

---

## 施設別を見る

**施設別_履歴** シートを使います。患者名は不要なので、そのまま施設名でフィルタ・集計できます。

---

## 注意（個人情報）

| OK | NG |
|----|-----|
| 患者マスタを秘書PCのみに保存 | 患者マスタを Google にアップロード |
| PID + 時間の共有シートを閲覧 | 患者名入りファイルをメールで送る |
| 【要登録】の PID をマスタに追加 | マスタを不特定多数と共有 |

---

## 困ったとき

| 症状 | 対処 |
|------|------|
| 全部【要登録】 | 患者マスタの PID 列が一致しているか確認 |
| シートがない | 医師に記録してもらう（初回記録で自動作成） |
| 今日の数字がおかしい | **集計_患者**（今日用）と **患者別_履歴**（履歴用）を混同していないか確認 |
